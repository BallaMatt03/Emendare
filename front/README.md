# Emendare - Front

## Project setup

``` shell
npm install
```

### Compiles and hot-reloads for development

``` shell
npm run serve
```

### Compiles and minifies for production

``` shell
npm run build
```

### Lints and fixes files

``` shell
npm run lint
```

### Run your unit tests

``` shell
npm run test:unit
```

### Run your end-to-end tests

``` shell
npm run test:e2e
```
